# Web Ads Monetization Guide (AdMob Alternative for Web)

Google **AdMob does NOT support web apps or websites**.  
This guide lists the **best ad networks for web** and explains **how to integrate them step by step**.

---

## Step-by-Step: Add Ads to a Web App

### Step 1: Prepare Your Website

Required pages:
- Privacy Policy
- Terms of Service
- Contact page

Ensure:
- Original content
- Clean UI
- No illegal content

---

### Step 2: Register With an Ad Network

Example (Google AdSense):
1. Create AdSense account
2. Add your website URL
3. Verify ownership (HTML tag or file)
4. Wait for approval

---

### Step 3: Create Ad Units

- Choose ad type
- Set size (responsive recommended)
- Copy ad code

---

### Step 4: Insert Ad Code Into Website

Common placements:
- Header
- Sidebar
- Between content
- Footer

```html
<script async src="https://adsbygoogle.js"></script>
